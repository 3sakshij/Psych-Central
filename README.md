# Psych-Central

The Aim of this project is to provide the Conversational online platform for psychotherapy and informative Website consist of various self-help guide which help the people suffering from mental illness to cope up with the their issues. To create Conversational Chatbot we make use of the Google Api called DialogFlow. This Api easily help to create the friendly virtual assitant that talks to user .
